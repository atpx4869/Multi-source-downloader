# 使用指南

## 快速开始

### 1. 安装依赖

**首次使用**：
```bash
双击运行: scripts\首次安装.bat
```

**或手动安装**：
```bash
pip install -r requirements.txt
```

**Windows 7用户**：
```bash
pip install -r requirements_win7.txt
```

### 2. 启动程序

```bash
双击运行: scripts\启动应用.bat
```

或清理缓存后启动：
```bash
双击运行: scripts\启动应用（清理缓存）.bat
```

## 核心功能

### 📥 多源数据下载

1. 在搜索框输入关键词（如"GB/T 3324"）
2. 选择要搜索的数据源（ZBY、GBW、BY）
3. 点击"搜索"按钮
4. 在结果列表中选择需要的文件
5. 点击"下载"按钮

**支持的数据源**：
- **ZBY** - 中标委数据源
- **GBW** - 国标网数据源
- **BY** - 标准分享网数据源

### 📋 标准文献查新

1. 点击"标准查新"选项卡
2. 上传包含标准号的Excel/CSV文件
3. 点击"开始查询"
4. 系统自动从多个数据源查询并补全信息
5. 查看结果（不同颜色标记不同状态）
6. 导出为Excel文件

**支持的信息**：
- 标准号
- 标准名称
- 发布日期
- 实施日期
- 状态（现行/废止/即将实施）
- 替代标准

### 🎯 批量下载

1. 在结果列表中按住Ctrl键多选
2. 或使用Shift键选择连续范围
3. 点击"批量下载"
4. 查看下载进度

## 配置说明

### API配置

编辑 `config/api_config.json` 配置数据源：

```json
{
  "zby": {
    "enabled": true,
    "timeout": 30
  },
  "gbw": {
    "enabled": true,
    "timeout": 30
  },
  "by": {
    "enabled": true,
    "timeout": 30
  }
}
```

详见 [config/API_CONFIG_GUIDE.md](../config/API_CONFIG_GUIDE.md)

### 下载目录

默认下载到 `downloads/` 目录，可在设置中修改。

### 缓存管理

缓存文件存储在 `cache/` 目录：
- 搜索结果缓存
- 下载历史记录

**清理缓存**：
```bash
双击运行: scripts\启动应用（清理缓存）.bat
```

## 常见问题

### Q: 程序启动缓慢？
A: 首次启动需要初始化，约2-3秒。如果持续缓慢，尝试清理缓存。

### Q: 搜索无结果？
A: 
1. 检查网络连接
2. 检查标准号格式是否正确
3. 尝试其他数据源
4. 增加超时时间（在配置文件中）

### Q: 下载失败？
A:
1. 检查网络连接
2. 尝试其他数据源
3. 检查磁盘空间
4. 查看日志了解详细错误

### Q: Excel导出失败？
A: 关闭其他打开的Excel文件后重试

### Q: 如何打包成exe？
A:
```bash
双击运行: scripts\打包程序.bat
```

生成的exe文件在 `dist/` 目录
